## Installation

Instructions étape par étape pour installer le projet localement.

1.  **Cloner le dépôt :**
    ```bash
    git clone https://github.com/Tedel12/educweb.git
    ```

2.  **Naviguer dans le répertoire du projet :**
    ```bash
    cd educweb
    ```

3.  **Installer les dépendances (exemple avec npm ou pip) :**
    ```bash
    npm install
    ```

## Utilisation


*   **Pour lancer l'application :**
    ```bash
    npm run dev
    ```
